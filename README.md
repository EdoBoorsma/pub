# Publicatie-repo (pub)

Deze repository bevat **uitsluitend statische publicatie-output** voor GitHub Pages.

## Belangrijk
- Inhoud in versie-mappen (`/vX.Y/`, `/specification/vX.Y/`) wordt **automatisch** gevuld door CI
  vanuit de `docs`- en `specs`-repositories.
- Wijzig deze mappen **niet handmatig**.
- Alleen de root-bestanden in deze repo zijn handmatig beheerd.

## Structuur
- `/` : landingpagina
- `/v6.0/`, `/v6.1/` : documentatie per minor-versie
- `/specification/v6.0/`, `/specification/v6.1/` : specificatie-documentatie per minor-versie
